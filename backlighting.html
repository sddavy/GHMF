Lighting

Use the table below to copy backlighting codes to use in the GHMF Software. Right-click on any square on the water timeline and select "Manual FCW Input" and paste the code into it, or press "W" and paste the code. Each module has five lights, placed in an "X", with one at each corner and one in the center facing up. In the software, turning on a module's lights controls all of these lights at once. However, using this table, you can control the back two lights in each module independantly, which is useful for lighting water features such as the wave without turning all the lights on.
